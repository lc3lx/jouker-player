# Tarneeb Test Server

سيرفر اختبار لطرنيب مع صفحة HTML/JS جاهزة للتجربة قبل ربط Flutter.

## التشغيل

```bash
cd backend
npm run test:tarneeb
```

**افتح المتصفح على:** http://localhost:3001  

(يجب فتح الصفحة عبر هذا الرابط وليس كملف محلي، لأن الاتصال يعتمد على WebSocket مع السيرفر.)

## ما توفره الصفحة

- **أوراق اللاعبين**: أوراقك ظاهرة، والخصوم كظهور ورقة (؟) مع **حجم اليد** لكل لاعب
- **الحركات**: Bid، Pass، Choose Trump، Play Card (بالنقر على الأوراق المسموحة فقط)
- **الأكلة الحالية (trick)**: الأوراق الملعوبة في الجولة
- **النقاط**: teamScores بعد كل جولة
- **الدور**: currentPlayerIndex مع تمييز اللاعب الحالي
- **رقم الجولة والحالة**: roundNumber و state (bidding / playing / round_end / game_end)
- **نهاية الجولة**: رسالة "End of Round" مع تفاصيل العقد واللفات، وزر **جولة تالية**
- **نهاية اللعبة**: الفريق الفائز والنقاط النهائية

## الاستخدام

1. اختر اللاعب (0–3) من القائمة
2. المزايدة: أدخل 7–13 ثم "مزايدة" أو "مرّ"
3. بعد الفوز بالمزايدة: اختر الحكم واضغط "اختيار الحكم"
4. في مرحلة اللعب: انقر على ورقة من "الأوراق المسموح لعبها" أو من يدك (المحددة بذهبي)
5. عند "End of Round": اضغط "جولة تالية"
6. "إعادة اللعبة": يبدأ توزيع جديد من الصفر

## الربط مع Flutter لاحقاً

- الأحداث: `get_state`, `action`, `reset_game`
- الردود: `game_state`, `action_result`, `round_result`, `game_finished`
- شكل الحركة: `{ playerIndex, action, payload }`
